# AI Furniture Matching System

## 🚀 Overview
This project builds an AI-powered system that helps users design furniture based on room dimensions, budget, and style preferences. It also estimates material and production cost.

## 🎯 Problem Statement
Furniture design is time-consuming and requires expertise. There is no simple system that suggests optimized furniture designs while considering space and budget constraints.

## 💡 Solution
We are developing an intelligent system that:
- Recommends furniture types based on room input
- Optimizes dimensions to fit space constraints
- Predicts cost of materials and construction
- Suggests design styles (modern, classic, minimal)

## 🧱 Tech Stack
- Python
- Machine Learning (Scikit-learn)
- FastAPI (Backend)
- Streamlit / React (Frontend)
